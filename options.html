Remote Reference Scripts Disabled
Some web pages with Bible references use a "remote script" ( RefTagger, etc.,) to create links of the references. PopVerse attempts to block these scripts so it can insert its links more effectivly. If you wish to view pages without PopVerse's influence, click off the Enable/Disable Button under the PopVerse icon (at upper right). Or you can do the same below.

Enable PopVerse. (Any open tabs must be refreshed to update.)

Link Partial References
Sometimes web pages have partial references; references that are missing the Bible Book (i.e 3:16). PopVerse can link these references using the Book from the last linked reference. Usually this works fine, however, it can also lead to wrong links on occasion. If you wish to enable this feature, check the box below.

Enable Partial Reference Linking. (Any open tabs must be refreshed to update.)

PopVerse Doesn't Appear To Work On All Pages

PopVerse should work on nearly all typical web pages where religious content is found, even most of those which have an existing "remote script" type linking system (such as gotquestions.org, etc.). We have tested PopVerse on many websites with various styles of layout and design, however, some highly specialized pages such as Facebook, gmail, etc., will not allow PopVerse to function. Usually this is a conflict with one or more of the many custom scripts currently running on the page.

Furthermore, if Bible references on a page are "hard linked" to another page (the references are tagged in the page source), PopVerse will not work. PopVerse works best on pages where the references are simple plain text.

If you encounter a page where PopVerse doesn't work and you think it should, contact us. We will check it out.


Guidelines For Entering References In PopVerse

The PopVerse Entry can accept nearly any standard Bible reference notation. The minimum consecutive characters needed to distinguish one book from another is all that is required,
  • Genesis 3:15 (or Gen. 3:15, Ge 3:15)
  • Psalms 2:10 (or Psa 2:10, Ps 2:10)
  • 1 Cor 1:5 (or 1Co 1:5)
Typical non-consectutive abbreviations can be used as well. Also, periods or colons can be used as chapter/verse separators,
  • Jn 3.16 (or Jn 3:16)
  • Mt 4.4
Verse spans and multiple verses within a single chapter can be included,
  • Mat 5:5-7
  • Luke 16:3, 5, 10
Chapters only and book names alone can be entered. With book names only the display will show the first chapter,,
  • Ge 3
  • Psa 23
  • Ruth (will display Ruth chapter 1)

Any entry beyond the range of existing chapters for a book will cause this page to be displayed. For example, Genesis has 50 chapters. If "Gen 55" is entered, there is no text to display.

Like many believers, we visit Christian websites on the Internet, some of them fairly frequently. Nearly always there is Scripture referenced in the text, often many references. However, when reading a page I almost never looked a reference up in my Bible.

Some sites have the the references as links which will take you to a webpage with the text or even show the text in a popup when the cursor is over the reference. This is a helpful feature, but there are still shortcominigs,

  • Often the Bible version you see is not the King James Version.
  • There is a delay before the popup displays, especially if you have a slow connection.
  • The links do not work offline at all.
  • You cannot enter a verse to view.
  • When you click a link you are taken to another site and often subjected to unwanted adds, promotions, etc.

Nevertheless, most sites don't have their Bible references as links, and that presented an opportunity.

After doing some research we determined a Chrome Browser Extension was the best direction to make an "stand-alone" reference tagger/viewer, and working in our spare time, the Lord allowed PopVerse to be developed. There is no other reference tagging tool like it, and here's why.

  • Fully Self-Contained; King James Version and Berean Standard Bibles are Included!
    PopVerse does not rely on online sources. The entire Bible text is included and access to it is instantaneous. It works perfectly offline.
  • Instant Reference Context Viewing
    Clicking a linked reference will cause a small "popup" browser window to open showing the entire Bible chapter containing the highlighted references(s).
  • Context Menu Verse List
    A Right-click will open a Verse List from References option where the text of all the Bible references on a page will be loaded in another popup browser window.
  • Easy Reference Entry
    A simple click of the PopVerse icon will open a Quick Reference Entry where the user can enter a reference to view.
  • View Key Cross-References from TSK
    The option is available to view up to four relevant cross-references per verse from the Treasury of Scripture Knowledge. The cross-references will display in the right column of the Chapter Window and will be live and interactive as other links.

PopVerse has proven to be a valuable tool for viewing Bible text from references in web pages.

Searching The Bible

Beginning with v2.0, PopVerse has had Bible searching capabilities. A user can search for a word, multiple words, or a phrase with optional wildcards.

Search terms can be entered in four ways:

  1. By entering the search query into the Entry Box under the PopVerse Extension Icon.
  2. By entering the search text into the Entry in the Search Panel under the Search tab.
  3. By selecting, dragging, and dropping a word from a page into the Entry Box.
  4. By selecting a word on a page and on right-click selecting "Search for word in Bible.".

How to perform searches:

  • To search for a single word, simply enter the word and click Search or press the Enter key.

  • If multiple words are entered separated by a space, PopVerse will search for ANY of the words. (A Boolean "OR" search).

  • If multiple words are entered separated by an " & " symbol, PopVerse will search for ALL of the words in a verse. (A Boolean "AND" search) For example born & again will only find verses that have both words in them.

  • If quotation marks are around words, the exact phrase will be searched for. i.e. "born again" will search only for that exact phrase. (Punctuation in the Bible text is ignored).

  • Wildcards can be used in conjunction with entered text. The Asterisk "*" acts as any unknown character(s). For example entering right* will find all words that begin with "right" ("righteous," etc.) Entering *right will find all words that end with "right" ("birthright," etc.

  • With the search results there is a concise multi-colored hit display that shows how the results are spread throughout the Bible. Each bar represents a Bible book and on mouseover will display the book name and number of search hits. When a bar is clicked the display will scroll to that book in the results list.

Hints:

  • Words found in the search results can themselves be used as search terms by right-click or by dragging and dropping into the Entry.

  • Wildcards can be used in combination with multiple word searches, i.e. etern* everl* will find "eternal," "eterninty," "everlasting," etc.
