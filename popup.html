PopVerse Bible Study System

PopVerse is a completely self-contained Bible study utility.

  • It automatically links Bible references found in webpages allowing the text to be viewed in a popup on mouseover or in a separate browser window on click.
  • It includes the Authorized King James and Berean Standard Bibles and works offline
  • It can show key cross references for each verse from the Treasury of Scripture Knowledge.
  • It will show a verse list from all the Bible references in the current page.
  • It has Bible searching capability with a visual search hit guide.
